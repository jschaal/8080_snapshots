 6510 SHEET TESTS — T098 through T110
 Add to the Unit Tests sheet starting at the next blank test slot.
 Each test follows the same layout as existing 8080 tests:
   Col A  = Test name (Unit_T098 etc.)
   Col B  = Hex Line  (0, 1, 2, 3...)
   Col C  = Dec Line  (auto-formula, same as other rows)
   Col D  = Label
   Col E  = OpCode
   Col F  = OP1
   Col G  = OP2
   Col H  = Expected Reg  (named range to check, or blank)
   Col I  = Expected Val  (expected value, or blank)
   Col J  = (leave as 0)
 Control table (cols P-T): set CPU = 6510 for all these tests.

Unit_T098  "6510 LDA/LDX/LDY immediate + flags"
────────────────────────────────────────────────────────────────────────────────
Tests that LDA/LDX/LDY set N and Z flags correctly.

  Hex  Label  OpCode  OP1   OP2   ExpReg  ExpVal
  0           LDA     #AA               
  1                               R_A     AA      ← A = $AA
  1                               N       1       ← bit7 set → negative
  1                               Z       0
  2           LDX     #01
  3                               R_X     01
  3                               N       0
  4           LDY     #00
  5                               R_Y     00
  5                               Z       1       ← zero flag set
  6           END

NOTE: Multiple assertions on the same row share the same Hex Line number.
Each assertion row has the Expected Reg + Expected Val filled in.
Only the FIRST assertion row for each line needs the OpCode filled in.

================================================================================
 ENTRY TIPS
================================================================================
 1. Copy an existing 8080 test block as a template (header row + data rows).
 2. Change the test name in col A to Unit_T098 etc.
 3. Fill cols B-I following the layout above.
    - Col B: Hex Line (0,1,2... in hex)
    - Col D: Label (only on rows that define a label)
    - Col E: OpCode
    - Col F: OP1
    - Col G: OP2 (leave blank for 1-operand instructions)
    - Col H: Expected Reg name (R_A, R_X, R_Y, N, Z, C, V, PC, SP etc.)
    - Col I: Expected value (hex for registers, 0 or 1 for flags)
 4. Rows with only H+I filled (no opcode) are pure assertion rows.
 5. Add to control table with CPU = 6510.
 6. Set Run = 1 to include in the suite.
